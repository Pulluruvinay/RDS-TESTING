# AWS RDS - User Information App

This repository contains a sample application that demonstrates how to build a simple web app using **Python Flask** and **AWS RDS (Relational Database Service)**.

The application collects user information from a form and stores it in an RDS database. It also retrieves stored user data and displays it through the web application.

This project shows how to:

- Use RDS for durable, scalable, and secure database storage

## Key Concepts Covered

- **AWS RDS Instance Setup**
- **Database connectivity from application**
- **Storing and fetching data using SQL**
- **Python Flask application architecture**
- **Security and configuration using environment variables**
